# RestaurantMenu

Console-based Java application to manage a restaurant menu, add percent-based special offers targeted to `food`, `drink` or `general`, and display the menu with or without applied discounts.

## Features
1. Manage menu items: add `Food` and `Drink`.
2. Add percent special offers scoped to `food`, `drink` or `general`.
3. Offers are added disabled by default; enable/disable offers manually.
4. Display menu showing best applicable discount per item and option to view prices without offers.

## Requirements
1. Java 17\+ (or Java 11\+)
2. Maven
